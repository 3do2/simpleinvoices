    • To use cron to generate recurrent invoices, prepare a file si_cron and place it in the /etc/cron.d folder in linux with the following contents: 
      # /etc/cron.d/si_cron: crontab fragment for simpleinvoices Recurrent Invoices Daily Cron
#  This creates recurring invoices for simpleinvoices once a day

#Simple Invoices recurrence - run each day at 1AM
#0 1 * * * /usr/bin/wget -q "http://www.si_domain.tld/index.php?module=api&view=cron"
